Displaying details about the single JSON object

Live example

JQuery loadJSON plugin enables you to bind a single object to the HTML template. This litterally represent loading JSON object into the HTML code. In the example below is shown how a single JSON object is bound to the HTML code

Implementation

HTML template that will be populated with JSON object should be defined. Elements in this template should have id attributes that matches properties of the object. Example is shown below:

        <label for="Name">Name:</label>
<span id="Name"></span>
					    <label for="Address">Address:</label>
<span id="Address"></span>
					    <label for="Country">Country:</label>
<span id="Country"></span>

<label for="Town">Town</label>
<span id="Town"></span>

<form action="edit.html" method="get">
   <input type="hidden" id="ID" />
   <input type="submit" value="Edit" class="submit-button"/>
</form>

JSON object that will be loaded into the HTML code shown above shoudl have properties that matches id attributes of the elements above. Example of such kind of JSON object is shown below:

        data = {
                    "ID":17,
                    "Name":"Emkay Entertainments",
                    "Address":"Nobel House, Regent Centre",
                    "Town":"Lothian",
                    "Contact":"Phone"
        }
